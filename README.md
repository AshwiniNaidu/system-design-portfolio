# system-design-portfolio 
This repository contains architecture case studies demonstrating
scalable system design, frontend platform architecture, and
enterprise architecture thinking.

Each case study includes:

* Problem statement
* Architecture diagrams
* Data design
* Scaling strategies
* Tradeoff analysis

---

## Case Studies

1. SaaS Multi-Tenant Platform
2. Large Scale E-commerce Platform
3. Real Time Collaboration System
4. Micro-Frontend Platform Architecture

---

## Architecture Focus Areas

- Scalability
- Microservices
- Event Driven Architecture
- Frontend Platform Design
- Cloud Native Systems

---

## Technologies Referenced

React, Node.js, Redis, Kafka, PostgreSQL, Kubernetes, CDN.
